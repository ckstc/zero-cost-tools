CSV 转 JSON 指南：在线互转方法

表格数据（CSV）和接口数据（JSON）经常要互转。本文讲清规则并给出零上传的互转工具。

CSV 转 JSON 怎么映射？

默认第一行为表头，作为每个对象的键；后续每行对应一个对象的值。例如表头 a,b 对应 {a:.., b:..}。

JSON 转 CSV 怎么处理？

取数组第一个对象的键作为表头，逐行写出各字段值；嵌套对象建议先展平。

为什么用本地工具？

业务数据常含敏感信息，本地处理可避免上传泄露风险。
