# 免费工具箱 / Zero-Cost Tools

零成本、注重隐私的纯前端在线工具合集。所有处理都在你的浏览器本地完成，图片/数据**不会上传到任何服务器**。

## 工具列表

| 工具 | 说明 | 链接（部署后） |
|---|---|---|
| 极速图压 | 图片压缩与格式转换（JPG/PNG/WebP），批量处理 | `/compress/` |
| JSON 格式化助手 | JSON 美化、校验、压缩 | `/jsonfmt/` |
| 免费工具箱 | 导航 Hub，交叉引流 | `/hub/` |

## 特点

- 🆓 完全免费，无需注册
- 🔒 隐私安全：纯前端运行，不上传数据
- ⚡ 零服务器成本：可托管于 GitHub Pages / Netlify / Vercel 免费额度
- 📱 响应式设计，移动端可用

## 本地运行

直接用浏览器打开对应 `index.html` 即可，无需任何依赖或构建步骤。
