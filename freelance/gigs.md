# 自由职业接单 — 可直接粘贴的简介与 Gig 模板（零成本）

> 用途：在 Upwork / Fiverr / 国内众包平台注册后，把下面模板改成你自己的信息即可发布。
> 原则：真实描述能力，不夸大、不虚假宣传、不刷单。

---

## A. Upwork 个人简介（Profile Summary）

```
I am a reliable freelance [写手/翻译/前端开发/数据标注] with [X] years of experience.
I deliver clean, on-time work and communicate clearly. I specialize in:
- [写作/翻译：中英互译、技术文章、产品文案]
- [开发：纯前端小工具、网页原型、自动化脚本]
- [数据：标注、清洗、表格整理]
No upfront payment required for small tasks — let's start with a trial.
```

## B. Fiverr Gig 示例（标题 + 描述）

**Gig 1：I will compress and optimize your images locally (privacy-safe)**
- 描述：Send me your images, I'll compress them with a local tool (no upload, privacy-safe) and return optimized files. Fast 24h delivery.
- 价格：Basic $5 / Standard $10 / Premium $20

**Gig 2：I will write a clear Chinese tech/article draft**
- 描述：You provide the topic/keywords, I deliver a structured, readable Chinese draft (800–1500 words).
- 价格：Basic $10 / Standard $25 / Premium $50

**Gig 3：I will build a simple free static web tool (HTML/CSS/JS)**
- 描述：A small client-side tool (calculator, converter, formatter) hosted free on GitHub Pages.
- 价格：Basic $30 / Standard $80

---

## C. 国内众包平台（百度众测 / 腾讯企鹅众测 / 阿里众包 / 京东微工）

- 注册后从「简单任务」起步：图片标注、语音转写、问卷；
- 不接需要「垫资」「刷单」「拉人头」的任务（红线）；
- 积累信用分后接更高单价任务。

---

## D. 接单安全守则

1. 只在平台内沟通与交易，不私下转账；
2. 不接「先交保证金/培训费」的活（99% 是骗局）；
3. 交付前留好沟通记录，纠纷走平台仲裁；
4. 遇到要求虚假评价、刷量的一律拒绝并举报。
